simplemail
==========

simple mail program.

Requirements
==============
- Python 2.4+

Usage
==============
- Message Body: standard input
- SMTP Host: OPTION(-H)
- SMTP Port: OPTION(-P)
- Mail Subject: OPTION(-s)
- Mail Attachemtn: OPTION(-a)
- Mail From: OPTION(-r)
- Mail To: arg(s)

for more details, run with OPTION '-h'
